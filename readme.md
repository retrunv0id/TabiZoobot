## Windows / VPS

1. Clone this repository
   ```
   git clone https://github.com/retrunv0id/TabiZoobot.git
   ```
2. goto TabiZoo directory
   ```
   cd TabiZoobot
   ```
3. install the require library
   ```
   pip install -r requirements.txt
   ```
4. execute the main program 
   ```
   python3 main.py
   ```
